# VisualSorting

Sorting algorithms animated using JavaScript.
You can test it out [here](http://www.debianserver.cz/www/VisualSorting/)

## Available algorithms

- Tim sort
- Insertion sort
- Selection sort
- Quicksort
- Merge sort
- Bubble sort
- Random sort
- Stalin sort
- LSD Radix sort
- MSD Radix sort
